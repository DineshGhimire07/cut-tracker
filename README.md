# CutTracker: Body Recomp OS

> **A premium, private command center for the disciplined.**  
> Not just a tracker. A decision-support system for fat loss.

---

## 🛡️ The Philosophy
Most fitness apps are built for social sharing or dopamine loops. **CutTracker** is built for data-driven results. It prioritizes:
- **Weekly Trends** over daily scale fluctuations.
- **Data Quality** over surface-level logging.
- **Analytical Honesty** over emotional reactions.

## 🚀 Core Features
- **14-Day Cycle Engine**: Focus on short, high-intensity tracking blocks.
- **Smart Analytics**: 8+ interactive charts visualizing weight trends (7-day rolling avg), macro adherence, step consistency, and more.
- **Cloud Sync**: Powered by **Supabase** for real-time syncing between desktop and mobile.
- **Decision Support**: Auto-generated weekly reviews that interpret your data to tell you if the cut is *actually* working.
- **Premium Aesthetics**: A custom-built, high-end dark interface designed for focus and performance.

## 🛠️ Tech Stack
- **Frontend**: React + Vite
- **Styling**: Pure CSS (Custom Design System)
- **Database**: Supabase (PostgreSQL + JSONB)
- **Charts**: Recharts
- **Icons**: Lucide React

## 📱 Mobile Access
This dashboard is fully responsive and deployed via **Vercel**. 
To access on your phone:
1. Open your Vercel URL.
2. "Add to Home Screen" for a native PWA-like experience.

## 🔑 Setup
1. Clone the repo.
2. Create a `.env.local` with your `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
3. Run `npm install` and `npm run dev`.
